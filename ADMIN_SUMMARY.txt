🎨 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Admin logs in at /admin with credentials
2. Admin makes changes in the control panel
3. Settings saved to localStorage
4. Custom event fires: 'adminSettingsChanged'
5. All user screens listen for this event
6. UI updates automatically without page reload

DATA FLOW:
  Admin → adminStorage.ts → localStorage → Custom Event → User Components
